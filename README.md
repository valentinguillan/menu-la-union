
# Menú Digital | Panadería & Confitería La Unión

Este es el sitio web del menú digital de **Panadería & Confitería La Unión**, diseñado para ser visualizado desde cualquier dispositivo móvil escaneando un código QR.

## 🧁 Características

- Diseño minimalista y moderno
- Navegación por secciones con fotos de productos
- Colores institucionales (#FFFFFF, #E74139, #F9E9D8, #3C3F62)
- Tipografía: Futura PT
- Imágenes de productos, con nombre, descripción editable y precio

## 📂 Estructura

```
├── index.html
├── logotipo.svg
├── isotipo-01.svg
└── productos_menu/
    ├── alfajores.png
    ├── brownie.png
    └── ... (más imágenes de productos)
```
